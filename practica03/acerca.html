Documentación Práctica 1

Trabajo realizado

Hemos realizado todos los apartados mencionados en el enunciado de la práctica, tanto las páginas HTML como el correspondiente estilo CSS según las indicaciones proporcionadas. Así, hemos hecho que todas las páginas de nuestro sitio web sean responsive, según la resolución de la pantalla donde se visualice. También hemos utilizado las etiquetas y atributos necesarios para validar algunos campos de los formulario de registro y login. Por último, hemos implementado la web utilizando Web Fonts e Icon Fonts tal y como se especificaba. Para que la web sea responsive hemos utilizado media queries y las instrucciones proporcionadas para realizar el menú hamburguesa únicamente con HTML y CSS. Tuvimos que diseñar diferentes encabezados según la resolución de la pantalla para su correcta visualización.


Incompatibilidades y problemas

Por lo general no encontramos demasiados problemas durante el desarrollo de la práctica. Tuvimos que investigar un poco especialmente a la hora de realizar las validaciones de los formularios y algunas animaciones que hemos incorporado como las de las fotos de las recetas o la del menú hamburguesa al aparecer y desaparecer. Sí que encontramos algún problema al probar la web en diferentes navegadores, especialmente en Mozilla Firefox, concretamente en dos aspectos. Por un lado, en algunos campos de formularios con números enteros, debido a que el alto del campo era demasiado pequeño, salían diferentes y para corregirlo, simplemente ampliamos el ancho de los campos (labels) del formulario. El otro problema que nos surgió y que nos llevó un poco más de tiempo resolver, fue que en las páginas con formularios, se cargaba el icono de la web en grande, y posteriormente la web hacía una especia de scroll y se situaba a mitad del contenido (en vez de arriba, como debería ser). Al final encontramos que el problema estaba en el icono ya que cargaba el html antes que el CSS, y le establecimos un tamaño fijo en el HTML para corregirlo.
